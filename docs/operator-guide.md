# Operator Guide

1. Confirm incorporation (Registre de Commerce et des Sociétés, Loi
   modifiée du 19 décembre 2002) and autorisation d'établissement (Loi
   modifiée du 2 septembre 2011 Arts. 3-5 — qualification,
   honorabilité, and a genuine fixed établissement that is NOT a mere
   domiciliation).
2. Intake against Portail des marchés publics (pmp.lu / marches.public.lu).
3. Jurisdiction assessment always requires human approval, even when
   the governor is clean.
4. Human-gated filing only — `:filing/draft`/`:filing/submit` never
   auto-commit, at any rollout phase.
