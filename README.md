# cloud-itonami-iso3166-lux

Open ISO 3166 Blueprint for **LUX**: Grand Duchy of Luxembourg.

Independent public-sector market-entry & procurement-compliance service
for an already-incorporated operator entering public contracts in Luxembourg.

## Official surface

- Procurement: Portail des marchés publics (pmp.lu) / public procurement portal
- Business/tax: RCSL commercial register + TVA registration
